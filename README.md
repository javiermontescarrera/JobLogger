# JobLogger

Precisiones acerca de la base de datos y la ruta de logs.

Base de Datos
•	Se ha incluido la base de datos BDLogger en la siguiente ruta:
~\utpJobLogger\BDLogger.mdf
•	Para hacer que el connectionstring (default) apunte a la instancia correcta, por favor, realizar los cambios en los archivos AppConfig incluidos en cada proyecto.
•	Adicionalmente, la clase respectiva permite también setear por código la base de datos.


Ruta por defecto Archivos log
•	Ruta por defecto: C:\Logs\
•	Si se desea cambiar la ruta por defecto donde se guardarán los archivos de texto generados por la aplicación, por favor, realizar los cambios en los archivos AppConfig incluidos en cada proyecto. 
•	Adicionalmente, la clase respectiva permite también setear por código la base de datos.
